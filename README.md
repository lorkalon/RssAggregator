# RssAggregator.
	Агрегатор Rss лент, с оффлайн хранилищем новостей.
	Краткое описание возможностей:
	* Добавление Rss лент(по прямому url, в будущем по базовому url сайта).
	* Удаление/редактирование добавленых лент.
	* Обновление лент.
	* Избранные новости.
	* Поиск в новостях(По ключевым словам, среди всех/не прочитаных)
	* Добавление тегов для новостей.

	* Открытие полной новости в окне браузера.
